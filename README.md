# Cybersecurity Lab Portfolio

This repository documents hands-on cybersecurity labs completed while developing practical skills in networking, threat analysis, vulnerability scanning, and security tool usage.

The goal of this portfolio is to build and demonstrate practical cybersecurity knowledge through lab exercises, tool experimentation, and capture-the-flag (CTF) challenges.

## Areas of Practice

Network Analysis  
Packet inspection and traffic investigation using tools like Wireshark and Nmap.

Vulnerability Scanning  
Identifying exposed services and potential security weaknesses.

Security Tool Usage  
Hands-on practice with common cybersecurity tools used in defensive and offensive security.

CTF Challenges  
Write-ups and solutions from cybersecurity challenge platforms.

## Tools Used

- Nmap
- Wireshark
- Burp Suite
- Metasploit
- Linux command line
